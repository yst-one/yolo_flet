# yolo-flet
YoloFlet control for Flet

## Installation

Add dependency to `pyproject.toml` of your Flet app:

* **Git dependency**

Link to git repository:

```
dependencies = [
  "yolo-flet @ git+https://github.com/MyGithubAccount/yolo-flet",
  "flet>=0.85.3",
]
```

* **uv/pip dependency**

If the package is published on pypi.org:

```
dependencies = [
  "yolo-flet",
  "flet>=0.85.3",
]
```

Build your app:
```
flet build apk -v
```
